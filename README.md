# Nostrpay

A Flutter app for kids to learn about Bitcoin.

## Setup

1. Install dependencies:

```bash
flutter pub get
```

2. Copy and configure app secrets:

```bash
cp lib/config/app_secrets.dart.example lib/config/app_secrets.dart
```

3. Edit `lib/config/app_secrets.dart` with your values.
